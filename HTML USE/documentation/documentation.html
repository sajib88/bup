Introduction

Espire is a unique WebApp template for admin and user dashboards with pixel perfect stylish design and tons of useful features. The package contains AngularJs and HTML5/jQuery version, which able to choose the best dashboard solution to fit your project needs. It is easy to customize and coded in developer friendly.

Thanks you for purchasing this template, If you have any question about this template, please feel free to contact us. We will provide our best assist.

Installation

Espire is built on top of Grunt and Bower tools. You may need to install the following things to get started. If your PC have installed nodeJS,Git, Grunt and Bower, you may skip this part

Install Node.js

Before installing Grunt, you must have NodeJs installed, Node.js® is a JavaScript runtime built on Chrome's V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js' package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

Download Node.js

Install Git

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

Download Git

Install Grunt

In one word: automation. The less work you have to do when performing repetitive tasks like minification, compilation, unit testing, linting, etc, the easier your job becomes. After you've configured it through a Gruntfile, a task runner can do most of that mundane work for you—and your team—with basically zero effort.

Run the command below at your cmd as Administrator to install Grunt

 npm install -g grunt-cli

Install Bower

Bower can manage components that contain HTML, CSS, JavaScript, fonts or even image files. Bower doesn’t concatenate or minify code or do anything else - it just installs the right versions of the packages you need and their dependencies.

Run the command below at your cmd as Administrator to install bower

npm install -g bower

Install Angular Cli (For Angular 5)

A command line interface for Angular

Run the command below at your cmd as Administrator to install Angular Cli

npm install -g @angular/cli

Command

Once you have completed the installation above, now you are ready to install the Espire.

Install node_modules & bower_ components

Navigate into the work folder which contain Gruntfile.js, run the following command and wait for it to finish:

npm install

You will found a node_modules folder were installed in your work folder

Run the following command as well at the same directory (Angular 5 users may skip this part)

bower install

a bower_components folder will be install in your work folder, which contains all the vendor files

If you encounter version choosing, please pick the latest version.

Task Command

After you successfully completed the 2 steps above, you will be able to run the following commands in your cmd from your work folder.

For Angular4 Version:
COMMAND DESCRIPTION
ng serve ng serve launches a localhost preview of your app, Navigate to http://localhost:4200/ to view result.
ng build ng build will compile all files into a folder called dist
Note: change <base href="/" > to <base href="./" >in dist/index.html after build done.
For Angular Version:
COMMAND DESCRIPTION
grunt live grunt live launches a localhost preview of your app (with Livereload) and will reflect all the changes you makes.
grunt sass grunt sass will compile the pages sass files to css on excute once.
grunt build grunt build wil minify your js & css into a folder called dist
For HTML Version:
COMMAND DESCRIPTION
grunt watch grunt watch his will automaically compile the sass files on save.
grunt sass grunt sass will compile the pages sass files to css on excute once.
grunt build grunt build wil minify your js & css into a folder called dist

Header Themes

We have already included 6 pre-made themes for header, Add the following css classes to body tag or <div class="app"> tag to customize your header.

  • .header-primary
  • .header-success
  • .header-info
  • .header-danger
  • .header-dark

Side Header

Switch your side header to dark by adding .side-nav-dark class to body tag or <div class="app"> tag.

RTL

Add .rtl class to body tag or <div class="app"> tag to active right to left layout.
